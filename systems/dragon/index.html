DragonHPC

Accelerate distributed Python workflows

Last updated: 22 Nov 2025     |     Release: 0.13

Dragon is a composable runtime that lets users create scalable, complex, and resilient HPC and AI applications and workflows through standard Python interfaces. Dragon provides capabilities to address many of the challenges around programmability, memory management, transparency, and efficiency on distributed systems.
